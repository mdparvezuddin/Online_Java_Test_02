Library Management System using OOP, Exception Handling & Collections

Problem Description:
Create a system that manages books in a library. Each book has Book ID, Title, Author, and
Availability status.

Tasks:
1. Class Design
Create a Book class with private attributes, constructors, getters/setters, and override toString().
2. Collection Usage 
Use a collection like ArrayList or HashMap to store books. Initialize with at least 5 books.
3. Functional Requirements 
Implement methods: addBook(), issueBook(), returnBook(), displayAvailableBooks().
4. Exception Handling 
Handle cases: book not found, already issued, invalid return. Create a custom exception
